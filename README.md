# Cloud Computing & DevOps Task 1

**Live URL:** [(https://d18hk77jqwnltz.cloudfront.net/)]

## Steps Taken:
1. **Local Development:** Built a responsive HTML/CSS portfolio with Google Fonts and FontAwesome.
2. **GitHub:** Initialized a Git repo and pushed the code to GitHub.
3. **AWS S3:** Created an S3 bucket (`pratik--portfolio`), enabled Static Website Hosting, and applied a public-read Bucket Policy.
4. **AWS CLI (or Console):** Uploaded the website files to the S3 bucket.
5. **AWS CloudFront:** Created a CloudFront distribution pointing to the S3 website endpoint, enforced HTTP to HTTPS redirection, and set `index.html` as the default root object.
